# Borough Bean Co
A modern, responsive cafe website that showcases menu itmes, location, and brand story for a fictional New York coffee shop.

## Live Demo
- Deployed site: https://bytemedarling.github.io/borough-bean-co/
- Repository: https://github.com/bytemedarling/borough-bean-co

## Features

- Responsive layout that works on desktop, tablet, and mobile.
- Multi-page navigation (home, menu, about, contact).
- Clear visual hierarchy using typography, color, and spacing.
- Semantic HTML for better accessibility and SEO basics.
- Simple animations/hover states to improve interactivity.

## Tech Stack

- HTML5
- CSS3 (flexbox, grid, media queries)
- GitHub Pages for hosting
